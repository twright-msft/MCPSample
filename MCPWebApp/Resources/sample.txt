This is a sample text file that demonstrates how the Model Context Protocol (MCP)

About MCP:
The Model Context Protocol is a universal, open protocol that enables seamless 
integration between AI tools and data sources. It standardizes how applications 
can expose their data and functionality to Large Language Models (LLMs) and 
other AI systems.

Key Features:
- Standardized interface for tools, resources, and prompts
- Language-agnostic protocol design
- Support for real-time data access
- Extensible architecture for custom integrations
